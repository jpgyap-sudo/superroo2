# HomeU Render Style Bible

Camera:

- 24mm to 35mm interior photography feel.
- Eye-level or slightly elevated angle.

Lighting:

- Soft daylight, warm practical lamps, realistic shadows.

Composition:

- Spacious, premium, clean.
- Product should be visible and central.

Forbidden:

- Wrong product replacement.
- Extra random furniture that conflicts with selected SKU.
- Distorted tables, chairs, legs, or stone veins.
