# Research and Answer Workflow

For consultation/research questions that require thorough analysis.

## Steps

### Step 1: Understand the Question
- Identify the core topic
- Determine what the user wants to know
- Note any specific constraints or context

### Step 2: Research Phase
Use your knowledge to:
1. Analyze the question from multiple perspectives
2. Consider industry best practices
3. Reference the SuperRoo architecture if relevant
4. Identify common pitfalls and trade-offs

### Step 3: Structure the Response
Organize your answer:
1. **Executive Summary** (2-3 sentences)
2. **Detailed Analysis** (key points with reasoning)
3. **Recommendations** (actionable advice)
4. **References** (if applicable)

### Step 4: Format for Telegram
- Use Markdown for readability
- Break into sections with *bold headers*
- Use bullet points for lists
- Keep under 4000 characters

### Step 5: Log for Learning
Record the Q&A pair as a training sample for the ML engine.
This helps the bot get smarter over time by learning which responses are most helpful.
