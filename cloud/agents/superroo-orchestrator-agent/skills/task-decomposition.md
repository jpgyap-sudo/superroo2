# Task Decomposition Skill

Break complex user requests into sequential subtasks:

1. Parse the user's goal into clear, independent steps.
2. Identify dependencies between steps.
3. Assign each step to the appropriate agent type (coder, debugger, tester, deployer).
4. Define success criteria for each subtask.
5. Set expected output format for each subtask.

Always prefer parallel execution for independent subtasks.
