# Result Aggregation Skill

Combine results from multiple agents into a coherent response:

1. Collect outputs from all completed subtasks.
2. Check each result for errors or warnings.
3. Merge code changes from multiple agents, resolving conflicts.
4. Summarize what was accomplished and what remains.
5. Present a unified status report to the user.

Flag any subtask that failed or was skipped for user attention.
