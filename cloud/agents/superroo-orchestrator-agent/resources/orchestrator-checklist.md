# Orchestrator Checklist

- [ ] User goal clearly defined.
- [ ] Task decomposed into sequential/parallel subtasks.
- [ ] Each subtask assigned to correct agent type.
- [ ] Dependencies between subtasks identified.
- [ ] All agents available and enabled.
- [ ] Results collected from all subtasks.
- [ ] Conflicts resolved in merged output.
- [ ] Final summary presented to user.
- [ ] Errors or skipped tasks flagged.
